¿Qué es HTML?

HTML son las siglas en inglés de HyperText Markup Lenguage, que significa Lenguaje de Marcado de HiperTexto. Se llama así al lenguaje de programación empleado en la elaboración de páginas Web, y que sirve como estándar de referencia para la codificación y estructuración de las mismas, a través de un código del mismo nombre (html).

Este código opera en base a la diferenciación y ubicación de los distintos elementos que componen la página Web. Así, el código es liviano y meramente textual, pero contiene las direcciones URL de las imágenes, audios, videos y otros contenidos que serán recuperados por el navegador para ensamblar la página, así como las indicaciones para la representación gráfica y estética del texto que se encuentre en la misma.

¿Para qué sirve HTML?

HTML se utiliza para estructurar el contenido en la web, como texto, imágenes y enlaces. Permite a los navegadores interpretar y mostrar el contenido de manera adecuada.

¿Cómo funciona HTML?

HTML usa etiquetas para definir la estructura del contenido. Los navegadores web leen estas etiquetas y muestran el contenido en la pantalla según la estructura definida.

Usos actuales de HTML

HTML se usa para crear páginas web y aplicaciones web. Es fundamental para construir el contenido de la mayoría de los sitios web modernos, desde blogs simples hasta aplicaciones web complejas.

Tags HTML

  • Definición y Función de las Etiquetas:Las etiquetas en el diseño de software se refieren a los textos que representan funciones en la interfaz de usuario, facilitando la identificación y navegación de los usuarios a través de la aplicación. Estas etiquetas actúan como un sistema de representación que utiliza términos o expresiones para identificar de manera inequívoca el contenido informativo.
  • Importancia del Etiquetado: El etiquetado ha ganado especial atención en la literatura académica y profesional, especialmente con el desarrollo de la web, debido al enorme volumen de información textual accesible a través de esta tecnología. Un etiquetado robusto requiere un análisis cualitativo y cuantitativo de los términos utilizados por los usuarios en sus contextos comunicativos e informativos.

Estructura de una Página Web

  • Componentes Principales: La estructura de una página web se organiza en varias partes fundamentales: encabezado, cuerpo y pie de página. El encabezado es crucial ya que es la carta de presentación de la marca y el primer punto de contacto con los visitantes. El cuerpo contiene el contenido principal, mientras que el pie de página incluye información adicional como datos de contacto y avisos legales.

Elementos Comunes de HTML

Los elementos básicos que se usan frecuentemente en HTML incluyen:

  • h1 a h6: Los encabezados en HTML se definen con las etiquetas `h1` a `h6`. El `h1` es el encabezado más importante y grande, mientras que el `h6` es el menos importante y más pequeño. Es recomendable usar solo un `h1` por página y no saltar niveles de encabezado para mantener una estructura lógica y accesible.
  • p: La etiqueta `p` se utiliza para definir párrafos en HTML. Los párrafos son bloques de texto que se muestran con un espacio antes y después del contenido, facilitando la lectura y organización del texto en la página.
  • pre: La etiqueta `pre` se usa para mostrar texto preformateado, es decir, el texto se muestra tal como se escribe en el código fuente, incluyendo espacios y saltos de línea. Esto es útil paramostrar código o texto que requiere un formato específico.
  • br: La etiqueta `br` se utiliza para insertar un salto de línea en el texto, creando una nueva línea sin iniciar un nuevo párrafo. Es útil para controlar el formato del texto en situaciones específicas.
  • hr: La etiqueta `br` se utiliza para insertar un salto de línea en el texto, creando una nueva línea sin iniciar un nuevo párrafo. Es útil para controlar el formato del texto en situaciones específicas.
  • a (href): La etiqueta `a` se utiliza para crear enlaces en HTML. El atributo `href` especifica la URL a la que apunta el enlace. Los enlaces son fundamentales para la navegación en la web.
  • img (src, width, height): La etiqueta `img` se usa para insertar imágenes en una página web. Los atributos `src`, `width`, y `height` especifican la fuente de la imagen, su ancho y su alto, respectivamente. Las imágenes enriquecen el contenido visual de la página.
  • style: El atributo `style` se utiliza para aplicar estilos CSS directamente a un elemento HTML. Esto incluye propiedades como `background-color`, `color`, `font-size`, y más. Los estilos permiten personalizar la apariencia de los elementos en la página.
  • sub y sup: Las etiquetas `sub` y `sup` se utilizan para mostrar texto en subíndice y superíndice, respectivamente. Estas etiquetas son útiles en contextos científicos y matemáticos para mostrar fórmulas y anotaciones.
  • small: La etiqueta `small` se usa para mostrar texto en un tamaño más pequeño que el texto circundante. Es útil para notas al pie, disclaimers, y otros textos secundarios.
  • i:La etiqueta `i` se utiliza para mostrar texto en cursiva. Aunque originalmente se usaba para enfatizar texto, hoy en día se recomienda usar `em` para ese propósito, reservando `i` para términos técnicos, nombres de especies, y otros usos específicos.
  • table: La etiqueta `table` se usa para crear tablas en HTML. Las tablas organizan datos en filas y columnas, y pueden incluir encabezados, cuerpos, y pies de tabla. Son útiles para mostrar información estructurada de manera clara y ordenada.
  • list:Las listas en HTML pueden ser ordenadas (`ol`) o desordenadas (`ul`). Las listas ordenadas muestran elementos en un orden numérico, mientras que las desordenadas usan viñetas. Las listas son útiles para agrupar elementos relacionados.
